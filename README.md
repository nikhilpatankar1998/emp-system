 Setup dodologs.
1. take clone from github repository.
 git clone https://github.com/travso-idice-2024/dodologs

2. change direcort to dodologs ->
cd dodologs

3. Install all dependency ->
composer install

4. Create .env file -> set db credentials.

5. create this folders ->

mkdir storage\framework\sessions
mkdir storage\framework\views
mkdir storage\framework\cache

6. Run this commands to server project ->
php artisan serve
